# BmanWeb
Personal Web site
# How to run
* Install nodejs
* Install git cli
* ` git clone https://github.com/branc116/BmanWeb `
* ` cd BmanWeb `
* ` npm install `
* ` npm run grunt `
* ` npm run start `
* Open loclhost:5555 in your browser
